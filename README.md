Javascript Accelerated
======================
This is a fast paced javascript course designed to get engineers up to speed with javascript programming.

Topics
------
* js History
* Primitives, Objects, and Gotchas
* Array Challenges
* js Frameworks
* Data
* Data Challenge
* Advanced Functions
* Advanced Arrays
* Building a jQuery Alarm Clock
